Basic commands used in Git

In my pc open vscode and open folder Git from D drive.
click on bottom left button with 2 zeros to open terminal.
cmd: cd 'current repository'(folder)
type the following command:git clone {file link} to open the file in vscode from github
type the following command:git status to dispaly the state of the code

  • if:up to date,then all commit are updated
  • if:untracked files,then the file is not saved in repository
  • if:modified,when we edited commited file in vscode
  • if:staged,then file is ready to commited
  • two steps
    1. first step: git add . (to stage all changes)
    2. second step: git commit -m " some amessage"(to save changes into repo)
  • cmd:git push origin main to push files from locla(from vscode) to Remote(Git hub)
